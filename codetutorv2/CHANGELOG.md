# Change Log

### Added 
- Major bug fixes
- Improvement to commands 
- Better responses

## [0.0.1] - 03/03/2026

### Added
- Initial release of Code Tutor V2
- Chat participant integration with GitHub Copilot (`@tutor`)
- **Explain Command** (`/explain`) - Provides difficulty-appropriate explanations of code concepts
- **Feedback Command** (`/feedback`) - Offers progressive feedback using the Socratic method
- **Exercise Command** (`/exercise`) - Generates programming exercises tailored to skill level
- **Level Command** (`/level`) - Allows students to adjust difficulty (1-4)
- Support for 11 programming languages: JavaScript, TypeScript, Python, Java, C#, C++, C, Go, Rust, PHP, Ruby
- Student profile tracking with difficulty levels and learning history
- Local storage of student metadata and progress data

### Features
- Adaptive explanations based on student difficulty level
- Context-aware feedback on student code
- Difficulty-calibrated exercise generation
- Progressive learning path support
- VS Code Chat API integration
- GitHub Copilot integration for AI-powered responses
